## What is Astro?

[Astro](https://astro.build) is a web framework designed for building content-driven websites like blogs, portfolios, and documentation.

### Key Features

1. **Zero JS by Default**: Astro ships zero JavaScript to the client by default, making your site incredibly fast.
2. **Islands Architecture**: You can interactive UI components (React, Vue, Svelte, etc.) only when needed.
3. **Content Collections**: Type-safe Markdown and MDX content management.

## Why use it?

If you care about performance and developer experience, Astro is a fantastic choice. It brings the best of static site generation with modern tooling.

```javascript
// Example of an Astro component
---
const name = "Astro";
---
<h1>Hello {name}!</h1>
```
