# Blog Post Images

This document tracks which images are needed for each blog post migrated from Ghost.

## Instructions

1. Export your Ghost content: Settings → Labs → Export
2. Extract the images from the export zip file
3. Copy images to `/public/images/posts/` directory
4. Images are already referenced in the markdown files

## Image Naming Convention

Use descriptive names with the post slug as prefix:
- `field-validation-error-example.png`
- `dataflows-mapping-screenshot.png`
- `command-bar-javascript.png`
- `hierarchy-visualization.png`
- `alm-accelerator-ui.png`

## Alternative: Direct URLs

If you can't export from Ghost, you can also:
1. Visit each post on josh-giles.ghost.io
2. Right-click images → Copy Image Address
3. Download images and save to `/public/images/posts/`
4. Images are already referenced in markdown files
