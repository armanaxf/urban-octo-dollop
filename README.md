# Minimal Astro Blog

A minimalistic blog starter built with Astro, Tailwind CSS, and Markdown.

## Features

- **Astro**: Fast, modern static site generator.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Markdown Content**: Blog posts are written in Markdown (MD/MDX).
- **Content Collections**: Type-safe content management.
- **RSS Feed**: Automatic RSS feed generation.
- **Netlify Ready**: Pre-configured for deployment on Netlify.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/content/posts`: Add your markdown blog posts here.
- `src/pages`: Route components (homepage, blog index, etc.).
- `src/layouts`: Page layouts.
- `src/components`: Reusable UI components.
- `astro.config.mjs`: Astro configuration.
- `tailwind.config.mjs`: Tailwind CSS configuration.
