# Blog Post Images Directory

Place your blog post images here.

## Instructions

1. Export content from your Ghost blog (Settings → Labs → Export)
2. Extract images from the export
3. Copy them to this directory
4. Images are automatically picked up by markdown files

## Naming Convention

Use descriptive names:
- `field-validation-example.png`
- `dataflows-configuration.png`
- `hierarchy-visualization.png`

## Supported Formats

- PNG (recommended for screenshots)
- JPG (for photos)
- WebP (for optimized images)
- GIF (for animations)
